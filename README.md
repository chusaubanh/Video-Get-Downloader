## ğŸ› ï¸ Build from Source

### Prerequisites

- **Node.js** 18.0.0 or higher
- **npm** 9.0.0 or higher
- **Git**

### Quick Build

```bash
# Clone the repository
git clone https://github.com/chusaubanh/Video-Downloader.git
cd Video-Downloader

# Install dependencies
npm install

# Download yt-dlp binary
# Windows:
curl -L -o electron/yt-dlp.exe https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe

# macOS:
curl -L -o electron/yt-dlp https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp_macos
chmod +x electron/yt-dlp

# Linux:
curl -L -o electron/yt-dlp https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp
chmod +x electron/yt-dlp

# Run in development mode
npm run electron:dev

# Build for your platform
npm run electron:build
```

### Build Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start web development server |
| `npm run electron:dev` | Start Electron with hot reload |
| `npm run electron:build` | Build for current platform |
| `npm run electron:build:win` | Build for Windows |
| `npm run electron:build:mac` | Build for macOS |
| `npm run electron:build:linux` | Build for Linux |
| `npm run electron:build:all` | Build for all platforms |

### Docker Build (Cross-Platform)

```bash
# Build for all platforms using Docker
docker-compose run builder

# Or specific platform
docker-compose run build-win
docker-compose run build-mac
docker-compose run build-linux
```

## â“ FAQ

<details>
<summary><b>Q: Why is the app blocked by antivirus?</b></summary>

This is a false positive. The app is open source and safe. You can:
1. Add an exception in your antivirus
2. Build from source to verify

</details>

<details>
<summary><b>Q: Download fails with "Video unavailable"</b></summary>

Possible causes:
- Video is private or deleted
- Region restriction
- Platform changed their API

Try updating yt-dlp in Settings.

</details>

<details>
<summary><b>Q: How to update yt-dlp?</b></summary>

The app bundles yt-dlp. To update:
1. Download latest from [yt-dlp releases](https://github.com/yt-dlp/yt-dlp/releases)
2. Replace `electron/yt-dlp.exe` (Windows) or `electron/yt-dlp` (macOS/Linux)

</details>

<details>
<summary><b>Q: Can I download playlists?</b></summary>

Currently, playlist download is not supported. Only single video download is available.

</details>
